
Перед запуском скриптов собирающих расширения, нужно проделать некоторые дополнительные шаги. А именно:

Для сборки расширения для Gogole Chrome:

1. Пакуем в .zip файлы из папки /extensions/chrome/ + d3.user.js
2. Заливаем в магазин Google Chrome получившийся .zip

Для сборки расширения для Mozilla FireFox:

1. Нужно поставить python (2.7 ok).
2. Mozilla Addon SDK.
3. Прописать пути к питону и папке SDK Расширений.
4. Запустить скрипт compileFirefoxExt.bat
